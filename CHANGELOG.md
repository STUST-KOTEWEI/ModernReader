## [1.0.0] - 2025-12-22

### Added
- 旗艦式 Hero 區塊 with 玻璃質感裝置預覽
- Signature Experiences section with 視差光暈效果
- Immersive Atelier 閱讀殿堂控制面板
  - 場景主題切換 (晨光/暮夜/琥珀)
  - 字體比例調整
  - 字距與呼吸控制
  - 沉浸音場選擇
- 策展星圖 (Curated Constellations) 橫向滑動收藏
- 禮賓服務模組 (Concierge Intelligence)
- LLM 智能聊天助手
- 預約導覽 Modal 系統
- 體驗模式 Demo
- 互動式動畫效果
  - Parallax scrolling
  - Card hover effects
  - Ripple effects
  - Cursor glow
  - 3D card flip effects
- 玻璃擬態設計 (Glassmorphism)
- Aurora 漸層背景效果
- Intersection Observer 進場動畫

### Technical
- HTML5 semantic markup
- CSS3 with custom properties
- Vanilla JavaScript (ES6+)
- Responsive design
- Google Fonts integration (Playfair Display + Inter)
- Mock API layer for future H.O.L.O. integration

### Documentation
- Comprehensive README with usage instructions
- Technical architecture documentation
- Connection to AI-Reader (Project H.O.L.O.) backend
- AI tool usage transparency statement
