# ModernReader Data Workspace

This workspace stores reproducible pipelines for corpus stewardship, augmentation, and evaluation.

- `ingestion/`: connectors and scripts for source collection (audio, text, metadata)
- `processing/`: cleaning, alignment, and CARE compliance filters
- `models/`: LoRA adapters, checkpoints, evaluation reports
- `catalogs/`: structured metadata exports for recommendation and catalog services
- `notebooks/`: exploratory analysis, prototyping, and reporting
