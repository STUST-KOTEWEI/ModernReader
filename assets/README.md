# Assets Directory

本目錄用於存放 ModernReader Royale 的靜態資源。

## 目錄結構

```
assets/
├── images/     # 圖片資源
│   └── README.md
├── icons/      # SVG 圖標
│   └── README.md
└── README.md   # 本文件
```

## 圖片規格建議

### Hero 圖片
- 建議尺寸：1920x1080px 或更大
- 格式：WebP (優先) 或 JPEG
- 壓縮品質：85%

### 書籍封面
- 建議尺寸：400x600px
- 格式：WebP 或 JPEG
- 比例：2:3

### 圖標
- 格式：SVG (首選)
- 大小：24x24px 基準
- 顏色：使用 currentColor 以便主題切換

## 命名規範

- 使用小寫字母
- 使用連字號分隔詞彙
- 描述性命名

範例：
- `hero-background.webp`
- `book-cover-stargazer.jpg`
- `icon-bookmark.svg`
