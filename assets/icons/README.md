# Icons Directory

本目錄用於存放 SVG 圖標。

## 建議圖標

- `bookmark.svg` - 書籤圖標
- `settings.svg` - 設置圖標
- `theme.svg` - 主題切換圖標
- `volume.svg` - 音量控制圖標
- `share.svg` - 分享圖標
- `menu.svg` - 選單圖標
- `close.svg` - 關閉圖標
- `search.svg` - 搜尋圖標

## SVG 最佳實踐

1. 移除不必要的元數據
2. 使用 `currentColor` 作為填充色
3. 設置 `viewBox` 屬性
4. 保持路徑簡潔
